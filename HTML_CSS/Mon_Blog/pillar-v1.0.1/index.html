  • Création d'un processus qui recueille des les données du capteur via le réseau sans fils
  • Création d'une base de données pour enregistrer les valeurs datés utilisant sqlite3 en C++

    • Sur l'Arduino :

  • Des tâches qui lisent périodiquement le capteur de lumière et qui l'envoie à la base
  • Afficher la valeur lue sur écran oled
